# Fractal Inspector

Preview and inspect components in Fractal component libraries.

## Installation

Install from NPM:

```
npm i @frctl/inspector
```

Then add a `inspector` config property to the Fractal configuration file within your project:

```js
// fractal.config.js
module.exports = {

  inspector: {
    // inspector config goes here...
  }

}
```

## Requirements

* Node >= v7.6
* Fractal >= v2.0
