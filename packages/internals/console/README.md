# Fractal Console

Formatted console log output for [Fractal](http://github.com/frctl/fractal) CLI commands.

[![NPM Version](https://img.shields.io/npm/v/@frctl/console.svg?style=flat-square)](https://www.npmjs.com/package/@frctl/utils)

## Installation

```
npm i @frctl/console
```

## Requirements

Requires Node >= v7.6
